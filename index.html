ENCRYPT EVERYTHING

A simple tool for encrypting and decrypting strings

Rotational Cipher is a simple substitutional cipher where each character is 'rotated' how ever many positions forward in the alphabet (ie: Using ROT13, 'a' becomes 'n').

Baconian Cipher changes each character in accordance to the baconian alphabet (ie: 'a' becomes 'aaaaa').

Rail Fence Cipher is a transpositional cipher where each character is written diagonally along 'rails' or rows. Once the maximum or minimum amount of rows is reached, the direction of the diagonal is changed. The final string is found by concatinating each row in order (ie: Using two rails, 'test' becomes 'tset')

Columnar Transposition Cipher is a transpositional cipher where a word is used as the key of the cipher. Each letter in the key is used as a column and each character in the string is assigned to a column. Once all the characters are assigned, the key is arranged in alphabetical order. The columns corresponding to each letter are concatinated together in this order (ie: Using the key 'test', 'word' becomes 'orwd').
